## MSYS2 简介

MSYS2 是一个在 Windows 上提供类 Unix 环境的工具集，包含 GCC、Make 等编译工具链，适用于需要命令行编译环境的开发场景。

## 安装

???+ note "注意"
    本文仅介绍 Windows 系统的安装步骤。

### 下载安装程序
访问 [MSYS2 官网](https://www.msys2.org/) 下载最新安装程序。

### 安装步骤
1. 运行安装程序
2. 按照默认选项完成安装
3. 安装完成后，从开始菜单运行 "MSYS2 UCRT64"

## 配置环境变量

为了在命令行中使用 MSYS2 的工具，需要将其添加到系统环境变量：

1. 打开系统属性 → 高级 → 环境变量
2. 在系统变量的 `Path` 中添加：`C:\msys64\ucrt64\bin`
3. 保存设置

## 测试安装

打开命令提示符，输入：
```bash
gcc --version

```如果显示 GCC 版本信息，则表示安装成功。

## 使用

你可以在记事本上编辑代码，在终点输入`g++ hello.cpp -o Hello`，就可以把名为`hello`的代码编译成`Hello.exe`了，其他语言类似，这里自行百度。

msys的另一用处就是配合其他软件使用，这里以 vscode 里的 c++ 为例。

1.安装汉化插件，C/C++ 插件。
2.用 vscode 打开一个 cpp 文件。
3.按`F1`，输入`> C/C++:选择 intelliSense 配置...`，选择默认编译器。
4.按`F1`，输入`> C/C++:编辑配置(UI)`，会弹出 c++ 插件的设置面板，选择你的编译喜好。
5.在 cpp 程序右上角找到一个虫子和三角形的符号，若点击可以正常调试，则配置成功。

其他语言同理。
